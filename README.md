# LeetCode Solutions

Welcome to my LeetCode problem solutions repository! This collection contains C++ implementations of various LeetCode problems, organized by difficulty and topic.

## 📚 Repository Contents

This repository contains solutions to LeetCode problems written in **C++**. Each solution is optimized for clarity and efficiency.

## 🏆 Problem Categories

Solutions are organized by:
- **Difficulty Level**: Easy, Medium, Hard
- **Topic**: Arrays, Strings, Trees, Graphs, Dynamic Programming, etc.

## 🛠️ Prerequisites

To compile and run the solutions, you need:
- C++ compiler (C++11 or higher)
- Basic knowledge of data structures and algorithms

## 💻 How to Use

1. **Clone the repository**:
   ```bash
   git clone https://github.com/Navraj88/LEETCODE.git
   cd LEETCODE
   ```

2. **Compile a solution**:
   ```bash
   g++ -o solution_name solution_name.cpp
   ```

3. **Run the solution**:
   ```bash
   ./solution_name
   ```
